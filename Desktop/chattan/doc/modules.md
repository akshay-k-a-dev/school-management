# Module documentation (module-wise)

This directory contains short, focused documentation for each major module in the project.

Files
- `auth.md` — authentication, JWT, password hashing
- `events.md` — event CRUD and registrations
- `bookings.md` — booking requests and admin approvals
- `complaints.md` — complaint submission, image uploads, admin actions
- `announcements.md` — create & list announcements
- `dashboard.md` — admin dashboard + export
- `profile.md` — profile view/update + change-password + my-events
- `models.md` — SQLAlchemy models and relationships
- `services.md` — business logic layer and responsibilities
- `routers.md` — router registration and page routes
- `frontend.md` — templates, JS and static assets

Open the specific file for endpoints, important implementation notes and places to change behavior.
